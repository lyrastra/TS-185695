# md-common-archive

## Владелец

Architecture Team

## Назначение

Набор классов для работы с архивами. Могут быть использованы в приложениях как на базе .net framework, так и на базе net core.

## Быстрый старт
1. Подключите сабмодуль:
   ```sh
   git submodule add https://gitlab.mdtest.org/development/infra/md-common-archive.git
   ```
2. Добавьте нужные проекты из папки `src/` в ваш solution (.sln):
   - Moedelo.Common.Archive

3. Вы готовы использовать инфраструктурный код для работы с архивами (см. документацию внутри классов).
