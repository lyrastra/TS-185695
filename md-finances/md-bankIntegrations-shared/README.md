# md-bankIntegrations-shared

**Владелец:** Команда интеграции

## Назначение
Общий модуль, реализующий базовую логику и инфраструктуру для взаимодействия с банковскими интеграциями в рамках платформы **bankIntegrations**.

## Основные возможности
- Единые интерфейсы и контракты для адаптеров банков.
- Унифицированные средства авторизации и обновления токенов.
- Базовые утилиты для работы с API и форматированием данных.
- Поддержка типовых сценариев обмена: получение балансов, операций, создание платежей.

## Совместимость и ограничения
- Используется совместно со всеми адаптерами банков (например, `md-bankAdapters-tinkoff`, `md-bankAdapters-point` и другими).
- Предполагает работу в составе сервисов платформы **bankIntegrations**.
- Не содержит бизнес-логики конкретных банков — только общий функционал и утилиты.
